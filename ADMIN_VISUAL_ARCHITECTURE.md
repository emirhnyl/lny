## 📈 Data Flow Diagram

```
┌─────────────┐
│   User      │
│  (Admin)    │
└──────┬──────┘
       │
       │ 1. Login Request
       ▼
┌──────────────────┐
│  NextAuth.js     │──────► Validate credentials
└──────┬───────────┘        Check permissions
       │                    Generate JWT
       │ 2. JWT Token
       ▼
┌──────────────────┐
│  Admin Panel     │
│   (Protected)    │
└──────┬───────────┘
       │
       │ 3. CRUD Operations
       ▼
┌──────────────────────────────────────┐
│         API Routes                    │
│  • Check JWT token                   │
│  • Verify permissions                │
│  • Validate input (Zod)              │
│  • Process request                   │
└──────┬───────────────────────────────┘
       │
       │ 4. Database Query
       ▼
┌──────────────────┐
│   Prisma ORM     │──────► Build SQL query
└──────┬───────────┘        Execute transaction
       │
       │ 5. Execute Query
       ▼
┌──────────────────┐
│   PostgreSQL     │──────► Store/Retrieve data
└──────┬───────────┘
       │
       │ 6. Return Data
       ▼
┌──────────────────┐
│   API Response   │
└──────┬───────────┘
       │
       │ 7. Update UI
       ▼
┌──────────────────┐
│  React Component │──────► Render new state
└──────────────────┘
```
